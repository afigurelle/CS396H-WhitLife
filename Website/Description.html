Project Description

Project Description: This app will have multiple features that will help new and old students connect, find helpful information, and interact with other students on campus. It will be an iOS application that will allow users to view local events at thead Whitworth campus, Primetime information, the Sodexo menu, and more features that will connect students to valuable information.

Features:

  • Events: Provide a schedule that is easy to read of upcoming events on the Whitworth campus. The list would update based on the schedule posted on the Whitworth website and would provide adequate information if each event is clicked on.
  • Sodexo: Allow users to see a dynamically updated daily menu for the Sodexo food service in the cafeteria.
  • Dorm Information: Provides basic descriptions and photos of each dorm for new and current students to view.
  • Primetimes: Features a centralized list of all primetimes for each dorm. RAs would be able to submit their week of primetimes to the app, and students would be able to view the information and location of each event.
  • Classifieds: Similar to the Whitworth Classifieds Facebook page, however easily accessible on mobile phones and private from Facebook.

Stretch Goals:

  • Profile: Allow user to create a personal profile to add friends and interact more personally with the various features of the app.
  • Chat: Messaging service that works within the app.

Device Requirements: iOS Application that can run on modern iPhone

Target Audience: Whitworth student body and visitng students
